영상을 Flask 프레임 워크로 계속해서 띄우려면 jsp로 웹을 구성하는게 맞을까요?
HTML은 화면에 cctv 스트림을 연속적으로 오픈하게 하는데 무리가 있을것 같습니다.

>
Flask 프레임워크를 사용하여 CCTV 스트림을 지속적으로 띄우는 것은 HTML과 JavaScript로 구성된 웹 페이지에서도 가능합니다. JSP(JavaServer Pages)은 Java 언어를 사용하여 동적 웹 페이지를 생성하는 기술이며, Flask는 Python 기반의 웹 프레임워크입니다. 따라서 Flask와 JSP를 함께 사용하는 것은 일반적으로 권장되지 않습니다.

대신, Flask 애플리케이션에서 HTML과 JavaScript를 사용하여 웹 페이지를 구성하고, CCTV 스트림을 지속적으로 띄울 수 있습니다. 이를 위해서는 HTML의 <video> 요소를 사용하여 CCTV 스트림을 재생하고, JavaScript를 사용하여 필요한 동작을 처리할 수 있습니다.

예를 들어, Flask 애플리케이션에서는 다음과 같이 HTML 템플릿을 사용하여 CCTV 스트림을 띄울 수 있습니다:

> 
이렇게 하면 Flask 애플리케이션에서 제공되는 /cctv_stream 엔드포인트로부터 CCTV 스트림을 받아와 <video> 요소에서 재생됩니다. Flask 애플리케이션에서는 해당 엔드포인트로부터 CCTV 스트림을 제공하는 라우트를 구현하면 됩니다.

이러한 방식으로 HTML과 JavaScript를 사용하여 Flask 애플리케이션에서 CCTV 스트림을 지속적으로 띄울 수 있으며, 별도의 JSP 사용은 필요하지 않습니다.
